# ⭐ 스타 코인 푸셔 (Star Coin Pusher)

브라우저에서 즐기는 인터랙티브 3D 오락실 코인 푸셔 게임입니다. 유튜브나 트위치 등의 라이브 스트리밍 시청자 참여형 게임(룰렛/뽑기 대용)으로 활용하기 좋도록 설계되었습니다!

## 🎮 주요 기능

- **실시간 3D 물리 엔진**: `@react-three/rapier`를 기반으로 한 사실적이고 묵직한 코인 충돌 및 낙하 물리 연산
- **채팅 기반 참여 시스템**: 스트리밍 채팅 내역을 복사/붙여넣기 하면 자동으로 참가자들의 이름을 추출해 각자의 코인을 생성해 줍니다.
- **다이내믹 셔플 박스**: 생성된 코인들을 상단 투명 박스에 모은 뒤, 섞기(`Shake`) 연출 후 한 번에 쏟아내는(`Pour`) 재미있는 연출 지원.
- **오락실 네온 테마**: 미려한 네온사인(핑크/시안/퍼플) 조명과 블러 처리된 유리 재질 등 레트로+모던 아케이드 감성의 디자인 적용.
- **1등 표시 & 로그 시스템**: 가장 먼저 앞으로 떨어지는 코인을 감지해 1등 시상 배너를 띄워주고, 낙하한 코인들의 기록을 UI에 실시간으로 표시.
- **배팅/피버 게이지 UI**: 기부/베팅 현황 및 피버 타임 연출을 위한 방송용 HUD UI 구비.

## 🛠 기술 스택

- **Core**: React 18
- **3D Rendering**: Three.js, React Three Fiber (@react-three/fiber), React Three Drei (@react-three/drei)
- **Physics**: Rapier 3D (@react-three/rapier)
- **Styling**: Vanilla CSS, Inline Styles, HTML CSS 오버레이
- **Build Tool**: Vite, TypeScript

## 🚀 로컬 실행 방법

1. 프로젝트 클론 및 의존성 설치
   ```bash
   git clone https://github.com/neverloselkb/coinpusher.git
   cd coinpusher-client
   npm install
   ```

2. 개발 서버 실행
   ```bash
   npm run dev
   ```

3. 브라우저에서 접속
   - `http://localhost:5173` 에 접속하여 확인바랍니다.

## 🕹 조작 방법 (단축키 및 기능)

1. **카메라 조작**: 
   - 마우스 왼쪽 클릭 + 드래그: 화면 회전
   - 마우스 우클릭 + 드래그: 화면 패닝(이동)
   - 마우스 휠: 줌 인 / 줌 아웃
2. **코인 생성 (Chat Panel)**
   - 왼쪽 중앙의 패널에 참가할 사람들의 이름을 줄바꿈으로 입력하거나 채팅창을 복사 붙여넣습니다. (중복 이름은 자동으로 1명으로 카운트됩니다)
   - 1인당 지급할 코인 개수를 설정 후 **"코인 생성"** 버튼을 누릅니다.
3. **셔플 및 투하**
   - 상단 박스에 코인이 쏟아지면 **"한번 섞기"**를 눌러 물리적으로 박스를 진동시킬 수 있습니다.
   - **"퍼붓기"** 버튼을 누르면 박스 바닥이 열리고 게임판으로 코인이 우르르 쏟아집니다!
4. **리셋 (새 게임)**
   - 좌측 중앙 UI의 우측 상단 **"새 게임"** 버튼을 누르면 게임판의 코인과 당첨 로그가 모두 초기화됩니다.
