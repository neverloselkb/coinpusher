# 🖥️ [치즈 코인 푸셔] 스트리머 제어판(Admin) UI 레이아웃

스트리머가 방송 전 세팅을 하거나, 방송 중 급하게 룰을 바꿀 때 헤매지 않도록 직관적이고 큼직하게 구성된 관리자 페이지 레이아웃입니다.

## 1. 📊 상단 대시보드 (상태 표시줄)

- **연결 상태:** `[🟢 치지직 서버 연동 중]` / `[🔴 연결 끊김]`
    
- **오버레이 URL:** `https://coinpusher.com/overlay/streamer123` 🔗**[복사하기]** (OBS 브라우저 소스에 넣을 주소)
    
- **실시간 통계:** 오늘 터진 황금 동전 갯수, 참여한 시청자 수, 현재 달성률 요약 표시.
    

## 2. 🎛️ 메인 컨트롤러 (원클릭 모드 전환)

제어판에서 가장 크고 눈에 띄는 중앙 영역입니다. 방송 흐름에 맞춰 즉각적인 조작이 가능합니다.

- **[ 🎲 가챠(도파민) 모드 ON ]** (버튼 클릭 시 보라색 네온 활성화)
    
- **[ 🤝 단합(영차영차) 모드 ON ]** (버튼 클릭 시 초록색 네온 활성화)
    
- **[ ⏸️ 동전 투하 일시정지 ]** (스트리머가 잠시 화장실을 가거나 룰을 설명할 때 사용)
    
- **[ 🔄 판 전체 초기화 ]** (내기가 끝나고 다음 판을 위해 화면 위 동전을 싹 치우는 비상 버튼)
    

## 3. ⚙️ 세부 난이도 설정 (아코디언 접이식 메뉴)

평소에는 접어두고, 필요할 때만 펼쳐서 세밀한 밸런스를 조절하는 영역입니다.

- **🎯 내기 목표 설정:** 목표 동전 갯수 입력 (예: 황금 동전 `[ 50 ]`개 떨어뜨리기)
    
- **물리 엔진 조절:** 푸셔(미는 기계) 왕복 속도 `[ 느림 / 보통 / 빠름 ]`
    
- **확률 커스텀:** 잭팟 등장 확률 `[ 10% ]` (슬라이더 바를 움직여 1~50%까지 스트리머가 직접 조절 가능)
    

## 4. 🎁 방송 준비 & 테스트 설정

- **가상 후원 테스트:** 방송 시작 전 OBS 화면에 잘 나오는지 확인하기 위한 `[ 🪙 1,000 치즈 가상 투하 ]` 테스트 버튼.
    
- **사운드 볼륨 (SFX):** 짤랑거리는 동전 소리, 대박 잭팟 브금(BGM)의 볼륨을 조절하는 슬라이더.
