# 🎰 [치즈 코인 푸셔] 확률 & 밸런스 모드 기획서

## 📌 개요
단일 확률로 인한 지루함을 방지하고, 방송의 목적(벌칙 내기, 시청자 단합, 텐션업 등)에 따라 스트리머가 직접 게임의 룰을 변경할 수 있는 **'투 트랙(Two-Track) 모드 시스템'**을 도입합니다.

---

## 🔥 모드 A: 가챠(도파민) 모드
**"누구 하나 얻어걸려라! 운빨 테스트"**
시청자 개개인의 운이 방송의 흐름을 결정짓는 짜릿한 모드입니다. 아슬아슬한 벌칙 내기나 심야의 하이텐션 방송에 적합합니다.

### 🎲 상세 확률 및 작동 방식
* **일반 채팅 (무과금 잭팟):** * `95% 확률`: 일반 은색 동전 1개 투하
  * `5% 확률`: **무료 황금 동전 1개 투하!** (시청자들의 '가즈아' 도배 유도)
* **치즈 후원 (과금 잭팟):**
  * 금액에 비례하여 묵직한 황금 동전 투하
  * `매 후원 시 10% 확률`: **'대박 피버 타임' 즉시 발동!** (수십 개의 동전이 비처럼 쏟아지며, 당첨된 시청자의 닉네임과 프로필이 화면 중앙에 크게 박제됨)

---

## 🤝 모드 B: 단합(영차영차) 모드
**"다 같이 밀어버려! 스노우볼 메타"**
한 명의 '큰손' 캐리보다는 시청자 전체의 화력이 모여야만 클리어할 수 있는 모드입니다. 방송 시간 연장 등 공동의 목표가 있을 때 시청자 참여를 극대화합니다.

### ⚙️ 상세 확률 및 작동 방식
* **일반 채팅 (화력 콤보):**
  * 화면 상단에 '단합 게이지' 생성
  * `조건`: 10초 이내에 서로 다른 시청자 30명이 채팅 입력 시
  * `효과`: 게이지가 100% 달성되며, 엄청나게 무거운 **'대형 무게추'**가 떨어져 판 위의 동전들을 시원하게 밀어냄.
* **치즈 후원 (글로벌 버프):**
  * 후원 금액에 따른 기본 동전 투하 외에 **전체 이로운 버프** 발동
  * `버프 1`: 10초간 기계 양옆에 **'가드레일'** 생성 (동전이 헛되이 옆으로 빠지는 것을 방지)
  * `버프 2`: 10초간 푸셔(미는 판)의 펌핑 속도 **2배 증가**

## 🖥️ 스트리머 제어판 (Admin UI) 적용안
스트리머가 방송 중에도 원클릭으로 흐름을 바꿀 수 있도록 직관적인 UI를 제공합니다.

* **모드 선택 토글 스위치:**
  * `[ 🔴 가챠(도파민) 모드 ]` ↔️ `[ 🔵 단합(영차영차) 모드 ]`
  * 클릭 시 실시간으로 웹 오버레이에 즉시 반영되며, 시청자들에게 "모드가 변경되었습니다!"라는 시각적/청각적 알림 제공.
* **세부 확률 조절 (선택 사항):**
  * 잭팟 확률(기본 10%)이나 단합 요구 채팅 수(기본 30개)를 스트리머의 시청자 규모에 맞게 커스텀할 수 있는 슬라이더 바 제공.

## 🎁 5. 특수 아이템 기믹 (Special Items)
단순한 동전 밀기를 넘어, 방송의 희로애락을 만들어낼 특수 아이템들입니다. 시청자의 특정 후원 금액이나 확률에 따라 등장합니다.

* 💣 **폭탄 (시청자의 트롤링 / 위기 유발)**
  * **효과:** 판 위에 떨어지면 3초 뒤 폭발하며, 주변의 동전들을 화면 밖(무효 처리)으로 날려버립니다. 
  * **활용:** 스트리머가 목표 달성을 코앞에 뒀을 때, 짓궂은 시청자가 후원하여 훼방을 놓는 용도입니다.
* 🛡️ **철벽 가드레일 (천사 시청자의 도움)**
  * **효과:** 10초 동안 기계 양옆의 구멍(Gutter)을 철조망으로 막아버립니다. 
  * **활용:** 밀려나는 모든 동전이 100% 득점으로 연결되는 '폭딜 타임'을 만들어 줍니다.
* 🧲 **블랙홀 자석 (대박 찬스)**
  * **효과:** 기계 끝부분에 생성되어 5초 동안 판 위의 모든 은동전을 강력하게 끌어당겨 한 번에 떨어뜨립니다. 황금 동전은 무거워서 천천히 끌려옵니다.
* 🧊 **얼음 덩어리 (스트리머 일시정지)**
  * **효과:** 거대한 얼음이 떨어져 푸셔(미는 기계)의 작동을 5초간 완전히 멈춰버립니다. 채팅이 아무리 올라와도 동전이 밀리지 않는 답답함(도파민 충전 전 단계)을 유발합니다.

## 🗓️ 6. 4주 완성 개발 로드맵 (Roadmap)
이 프로젝트를 가장 빠르고 효율적으로 런칭하기 위한 주차별 스케줄입니다.

### 🏃 1주차: 핵심 물리 엔진 및 3D 프로토타입
* `Three.js`와 `Cannon.js`(물리엔진) 세팅.
* 투명한 유리관과 움직이는 푸셔(판) 3D 모델링 구현.
* 클릭 시 동전이 떨어지고 자연스럽게 밀려나는 기초 물리 법칙 최적화.

### 🔌 2주차: 치지직 API 연동 및 데이터 파이프라인
* 치지직 비공식 API 또는 WebSocket을 연결하여 라이브 채팅 데이터 파싱.
* 특정 채팅(예: `!참가`) 및 일반 채팅 입력 시 은동전 낙하 로직 연결.
* 치즈(후원) 데이터 수신 시 황금 동전 및 파티클 이펙트 연결.

### 🎛️ 3주차: 관리자 페이지(Admin) 및 투 트랙 모드 구현
* 스트리머가 로그인해서 설정값을 바꿀 수 있는 Admin UI 개발.
* 기획서에 명시된 `가챠 모드`와 `단합 모드` 토글 스위치 및 로직 적용.
* 내기 목표 설정 (예: 황금 동전 50개) 및 프로그레스 바 UI 오버레이 작업.

### 🎬 4주차: OBS 송출 테스트 및 폴리싱(마무리)
* 실제 OBS 브라우저 소스로 불러와 투명 배경 및 해상도(1920x1080) 깨짐 확인.
* 동전이 떨어질 때의 사운드 이펙트(SFX) 및 피버 타임 시각 효과(네온사인) 추가.
* 메모리 누수(동전이 너무 많아져 방송이 끊기는 현상) 방지를 위한 최적화 및 최종 버그 픽스.


**💡 작성일:** 2026년 2월
**💡 활용 방안:** 치지직 스트리머 대상 참여형 콘텐츠 및 후원 유도
